# iMaclaw Website

iMaclaw 是 IMA Studio 推出的 AI 智能助手产品，基于 OpenClaw 架构，为用户提供开箱即用的 AI 内容创作体验。

## 项目结构

```
imaclaw-website/
├── index.html          # 主页面
├── css/
│   └── styles.css      # 自定义样式
├── js/
│   └── main.js         # 交互逻辑
└── components/         # React Bits 动效组件
    ├── Aurora/         # 极光背景
    ├── OrbitImages/    # 轨道图片
    ├── SplitText/      # 文字拆分动画
    └── ...
```

## 设计理念

- **科技感**：深色主题 + 霓虹渐变
- **灵动性**：React Bits 动效组件
- **简洁性**：信息层级清晰，减少认知负担

## 技术栈

- HTML5 + CSS3
- React Bits 动效组件
- Framer Motion (via CDN)
